How to Play :

(Mario) has to push every crate into an objective cell Winning is when all crates are in objective cells.


Hotkeys :

Movement : Arrow Key.


How to use the Editor :

The editor is a tool to make your own map and play it.

  • Left Click : to replace a block.

  • Right Click : to remove a block.

  • ESCAPE : to save the map.

  • To switch the block type :

    • 1: Wall.
    • 2: Crate.
    • 3: Objective.
    • 4: Mario.
